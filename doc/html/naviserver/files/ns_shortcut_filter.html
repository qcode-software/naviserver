ns_shortcut_filter(n) 4.99.23 naviserver "NaviServer Built-in Commands"

Name

ns_shortcut_filter - Stop processing url filters

Table Of Contents

  • Table Of Contents
  • Synopsis
  • Description
  • COMMANDS
  • See Also
  • Keywords

Synopsis

  • ns_shortcut_filter when method urlPattern

Description

Other filters that also match when+method+urlPattern will not run after this filter.

COMMANDS

ns_shortcut_filter when method urlPattern

See Also

ns_register, ns_register_filter, nsd
